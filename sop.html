02.3 / Procédures Standards

Standard Operating Procedures (SOPs)

Les 12 Étapes du Pipeline StrategyLab

SOP-01 : Prospection Outbound

Responsable : John

Procédure : Extraction hebdomadaire de 250 leads qualifiés sur LinkedIn Sales Navigator. Critères : Fondateurs startups (Seed/Series A) ou DSI/PME en Afrique de l'Ouest. Nettoyage via Dropcontact/NeverBounce (bounce < 1%). Injection dans Smartlead. Séquence de 3 emails espacés de 4 jours, orientation vers Cal.com pour RDV.

SOP-02 : Découverte et Qualification

Responsable : John / Rodolph

Procédure : Appel découverte avec grille BANT (Budget, Authority, Need, Timeline). Si budget < 3000 USD ou décision tierce, lead = "Non Qualifié".

SOP-03 : Établissement du Devis

Responsable : Rodolph

Procédure : Après validation technique par Hanss, rédaction proposition commerciale. Coefficient sécurité 1.3 appliqué sur estimations temps.

SOP-04 : Contractualisation Juridique

Responsable : Rodolph

Procédure : Génération contrat type. Envoi via Dropbox Sign. Règle d'Or : Aucun travail ne démarre avant signature et acompte 50% reçu.

SOP-05 : Branding & Identité

Responsable : Herman

Procédure : Envoi brief créatif automatisé. Deux moodboards/logos. Présentation 45 min. Développement Brandbook (logo, typo, layouts).

SOP-06 : Design UI/UX

Responsable : Herman

Procédure : Wireframes Lo-Fi → validation → Hi-Fi sur Figma. Design System interne (Auto-Layout, Tokens, Composants). Prototype cliquable.

SOP-07 : Développement Logiciel

Responsable : Hanss / Jacques

Procédure : Dépôt GitHub privé. GitFlow (main=prod, staging=tests, develop=intégration). Branches feature/* pour Jacques. Pull Request obligatoire validée par Hanss.

SOP-08 : Assurance Qualité

Responsable : Hanss

Procédure : Tests Jest/Cypress automatisés. Audit Google Lighthouse. Scores min requis : Accessibilité/SEO/Pratiques/Vitesse = 90/100.

SOP-09 : Déploiement & Livraison

Responsable : Hanss

Procédure : CI/CD vers Vercel/AWS/Railway. Certificats SSL. DNS redirection. Signature PV déclenchant facturation solde.

SOP-10 : Support & Maintenance

Responsable : Jacques

Procédure : Intégration client support Discord. Incident critique : 2h max. Requêtes mineures : 48h ouvrées.

SOP-11 : Facturation & Recouvrement

Responsable : Rodolph

Procédure : Facture Quickbooks à chaque jalon. Relance J+2. Appel D+7 si non-paiement. Mise en demeure J+15.

SOP-12 : Archivage & Capitalisation

Responsable : Tous

Procédure : Repo GitHub en Read-Only si pas de maintenance. Nettoyage Figma. Archivage Google Drive chiffré. Post-mortem interne 30 min.
